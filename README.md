# Fucking-Fighter pv-bot

This is my bot for Payvision Bots League.

## Build dev

1. npm install
2. npm start -> this will create a web server in localhost:3000

## Deploy with docker
1. docker build . -t pv-bot
2. docker run -d -p 3000:3000 pv-bot:latest
